# First Name Generator
A simple place to generate first name

## 💫 What does it do?
It generates a random popular first name when you click generate button.

## ⚙️ How does it work?
It works by generating a random decimal between 0 (inclusive) and 1 (exclusive) upon clicking the "generate" button. Multiplying it by the array's length scales it to the array's index range. It is then rounded down the result to an integer, which is then used as the array index. Finally, the selected name will be displayed on the web.
